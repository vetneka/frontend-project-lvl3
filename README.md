> study project at hexlet.io

# Installation and Usage
### Prerequisites:
```
node.js >= 14
npm >= 6
make >= 4
```

You can install **rss agregator** using *make*:
```
$ make install // install dependencies
```

# Run tests

```
$ make test
```
